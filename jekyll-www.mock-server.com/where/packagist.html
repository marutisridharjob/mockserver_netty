There is a PHP integration for mockserver hosted on github: lequipefr/mockserver-behat-context.

This library provides:

  • a Behat context for functional testing,
  • a PHP client to send raw expectations as array,
  • and a builder to use autocompleted methods instead of arrays.

Install from packagist

The package is hosted on Packagist: lequipefr/mockserver-behat-context.

Install it in your project as a dev dependency: 

composer require --dev lequipefr/mockserver-behat-context

Follow the README for more information about installation and use: README lequipefr/mockserver-behat-context.
